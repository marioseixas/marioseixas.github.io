# Transferindo o dinheiro
1. Nas imagens acima, verificar o valor correspondente ao mês corrente a ser repassado para o Le
2. Transferir esse valor do BRB para o NuBank por meio de pagamento do boleto gerado pelo aplicativo do NuBank
3. Após um dia útil do pagamento do boleto, verificar se o dinheiro está na NuConta
4. Fazer uma TED pelo aplicativo do NuBank da NuConta para a conta corrente do Le 
5. Enviar comprovante da transferência para o Le via WhatsApp
